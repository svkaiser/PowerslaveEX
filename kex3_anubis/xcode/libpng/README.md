libpng-xcode-project
====================

Xcode project of libpng for Apple OSX / iOS.

Tested on v1.6.13.

Installation
-------------
1. Download libpng from official repository http://libpng.sourceforge.net;
2. Unarhive;
3. Put Xcode folder to ./projects folder;
4. Build.
